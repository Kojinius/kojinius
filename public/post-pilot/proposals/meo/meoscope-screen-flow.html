分析 1 ジョブの動線
L0 ログイン→ S1 分析開始 (name/map/area)→ (候補複数なら S1.5 /select→) S2 /analyze SSE 8 ステップ→ S3 /dashboard/[id]→ S4-P /print (Ctrl+P) もしくは S4-X /api/report/excel→ DM / 提案書に添付
再閲覧の動線
L0 ログイン→ S5 /history→ 行クリック→ S3 /dashboard/[id]→ 必要なら S4 出力
本資料の対象外 管理用ルート (/admin · /admin/users) と内部 API ルート (/api/analyze · /api/report/pdf 等) は遷移図から除外。エンドユーザー(=営業担当)が日常的に踏む画面のみを抜粋している。
