よくあるご質問

心配事は、ここで解消できます

直接 SNS アプリを開けばいいのでは？
違いは「下書きが目の前にあるかどうか」です。「何を書こう」で迷う時間が消えるので、投稿の継続率がまったく違います。3案から選ぶだけで、業種・季節に合った投稿が10秒で完了します。
本当に無料ですか？ あとから請求が来ませんか？
完全無料です。クレジットカード登録もありません。請求書をお送りすることもありません。
SNS のアカウント情報を渡さなくて大丈夫ですか？
大丈夫です。このシステムは、あなたのアカウントに勝手にアクセスすることはできません。投稿の最終決定は、必ずあなたが SNS 公式アプリで「投稿」ボタンを押した瞬間に確定します。
X や Threads のログインは必要ですか？
投稿時には、X / Threads それぞれの公式アプリ（またはブラウザ）にログイン済みである必要があります。これは普段から SNS を使っている方であれば、ほぼ常にログイン状態のはずなので追加の操作は発生しません。万が一ログアウト状態だった場合は、ボタンを押したあとに各 SNS のログイン画面が表示され、ログイン後に下書き入りの投稿フォームが開きます。

当サービス側へのログイン・会員登録は 一切不要 です。SNS 側のログインは投稿の最終確定のために 必要です。ここを区別してご理解ください。
下書きはどうやって作られていますか？
業種テンプレ × 季節カレンダー × トレンド情報を組み合わせて、人の目を通したうえで毎日ご提供します。「カフェ向け」「美容室向け」など業種ごとに専用テンプレが用意されています。
過去30日と被らないって、どうやって判定するの？
あなたが押した投稿ボタンの履歴を、ブラウザ内に記録します（サーバには送りません）。次回以降の下書き表示時に自動で照合し、被る案は別案に差し替えます。履歴はいつでもクリアできます。
通知リマインダって、勝手に通知が来るの？
初回に「通知を許可するか」確認してから始まります。許可しなければ通知は一切来ません。許可したあとも、いつでも解除できます。
予約投稿（〇時に自動で投稿）はできますか？
簡単投稿プランは "ボタンを押した瞬間" の投稿に特化しています。決まった時間に自動配信したい場合は、姉妹プランの「盛り盛りMAXプラン」をご覧ください。
LINE やジモティへの投稿はできますか？
簡単投稿プランの対応媒体は X / Threads の2つです。ジモティは提供側で外部投稿の仕組みが廃止されたため対応していません。LINE 公式アカウントから友だちへ配信したい場合は、姉妹の「盛り盛りMAXプラン」で LINE Messaging API を使った公式アカウント連携をご利用ください（LINE 公式アカウントの月額料金が別途必要です）。
文字数の上限はありますか？
あります。X は全角を2文字と数える加重で280まで、Threads は500文字までです。本文＋ハッシュタグがこの上限を超えると、その媒体の投稿ボタンは自動的に無効になり、ボタン下のカウンタが赤く表示されます。下書きを少し短くすれば投稿できます。
